# Lab Status Dashboard

This is the laboratory command center for the Chrono Rig.

## Neural Architecture
- **Host:** the_host-mlrig
- **GPU:** NVIDIA RTX 6000 (Blackwell) - 96GB VRAM
- **OS:** Ubuntu 22.04 LTS

## Functional Modules
- **Thermal Core:** Real-time GPU temperature monitoring.
- **Brain Load:** GPU utilization and power draw tracking.
- **Model Loadout:** Active model monitoring and VRAM usage.
- **Neural Dreaming:** Subconscious knowledge synthesis engine.
- **Rig 3D Visualizer:** Real-time 3D model of the lab hardware (Experimental).

## Access
- **Port:** 8889
- **Internal URL:** http://localhost:8889
- **Dashboard Commlink:** Integrated WebSocket chat for direct AI interaction.
